# Exports Directory

Direktori ini menyimpan file hasil export laporan.

File-file export akan otomatis dihapus setelah beberapa waktu untuk menghemat ruang penyimpanan.

**Format File:**

- PDF: Untuk tampilan yang siap cetak
- CSV: Untuk analisis data di Excel/Google Sheets

**Struktur Nama File:**

- laporan_YYYY-MM-DD_HH-MM-SS.pdf
- laporan_YYYY-MM-DD_HH-MM-SS.csv

**Konten Export:**

- Laporan Pendapatan
- Laporan Pengeluaran
- Data Pelanggan
- Data Kinerja Karyawan
